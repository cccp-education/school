= Databases

== Postgresql

.start
[source,bash]
----
sudo docker compose -f ~/workspace/atelier/database/postgresql/postgresql.yml up -d
----

.stop
[source,bash]
----
sudo docker compose -f ~/workspace/atelier/database/postgresql/postgresql.yml down
----
